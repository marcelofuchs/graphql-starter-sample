# Graphql starter sample

## Installing...

### Docker Compose:

* Install docker: 
    - https://docs.docker.com/install/
    - https://get.docker.com/
* Install docker-compose
    - https://docs.docker.com/compose/install/
* Run docker-compose:

```
$ docker-compose build
$ docker-compose up
```
* Access: http://0.0.0.0:5000/
